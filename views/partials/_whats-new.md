<h3 class="govuk-heading-s">16 July 2026: We’ve released GOV.UK Frontend v6.4.0.</h3>
<p class="govuk-body">This release adds an <a href="/components/panel/#interruption-panel" class="govuk-link">interruption variant of the Panel component</a>, which you can use to pause the user’s journey and give them important information.</p>
<p class="govuk-body">We’ve also improved the Date input component and introduced a new <code>inverse-text</code> functional colour to use on dark backgrounds.</p>

or <a href="/community/whats-new/" class="govuk-link">see our latest updates on the What's new page</a>.</p>
